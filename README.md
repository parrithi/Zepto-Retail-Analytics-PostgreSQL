# 🛒 Zepto Inventory Analysis using PostgreSQL

## 📌 Overview

This project analyzes Zepto's inventory dataset using PostgreSQL to uncover insights related to product pricing, discounts, inventory availability, and category performance.

## 📊 Dataset

* Source: Kaggle - Zepto Inventory Dataset
* Records: 3,700+ products
* Columns: Product details, pricing, discounts, inventory status, and categories

## 🛠 Tools Used

* PostgreSQL
* SQL
* GitHub

## 🔍 Analysis Performed

* Data Cleaning
* Inventory Analysis
* Pricing Analysis
* Discount Analysis
* Revenue Estimation
* Category Performance Analysis

## 📈 Key Insights

* Identified high-value products that were out of stock.
* Analyzed discount trends across product categories.
* Estimated inventory value and revenue opportunities.
* Compared pricing and product availability across categories.

## 💡 SQL Concepts Used

* SELECT, WHERE
* GROUP BY, ORDER BY
* Aggregate Functions
* CASE Statements
* Subqueries
* Business KPI Calculations

## 📂 Project Files

* `zepto_v2.csv` – Dataset

* `README.md` – Project Documentation
